PUMA 3D Printing Files
======================

This folder contains information about printing STL files associated witht he PUMA microscope project. 

The folder CE3_Cura provides Cura profiles that are suitable for the Creality Ender3 printer using standard PLA filament. If you don't use that printer or filament type then you would need to find your own 'best' profiles for printing the models.

The information given the the PUMA 3D printing instructions relate to prints made with the Ender3. Other printers may require modifications of the advice given.
